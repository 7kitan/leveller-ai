# Project Mission: AI CV Processor & Inference Hub

## Objective
Modernize the CV processing pipeline by replacing legacy local extraction logic with a scalable, API-based architecture powered by Chandra OCR and semantic BERTScore analysis.

## Core Capabilities
1. **Dynamic OCR**: Robust document parsing using specialized VLMs.
2. **Semantic Matching**: High-precision skill gap analysis using BERTScore.
3. **Agentic Automation**: Full lifecycle management via Antigravity hooks and rules.
